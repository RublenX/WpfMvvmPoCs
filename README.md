# WpfMvvmPoCs
Reciclando conocimientos de MVVM sobre WPF

# Ejemplo de Organización de Carpetas
- src: para el código fuente
- test: para pruebas unitarias
- resources: para imágenes y archivos de recursos
- docs: para la documentación del proyecto
- libs: para bibliotecas y paquetes
- build: para los ficheros compilados o de entrega
- public: para publicar el proyecto

# Proyecto WpfPoCs
Es un proyecto sin funcionalidad con la única intención de reciclar los conocimientos que tengo de WPF y el patrón MVVM en las nuevas versiones, ya que vengo de MVVMLigth.
